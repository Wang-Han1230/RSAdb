RSAdb

Single-cell omics big data platform for recurrent spontaneous abortion

Welcome to RSAdb

RSAdb is a comprehensive open-access resource for investigating transcriptomic alterations associated with gene expression patterns within single-cell omics datasets at the maternal-fetal interface in human recurrent spontaneous abortion (RSA).

View on GitHub Download Code

Features

Interactive Exploration

Explore 25,858 genes across 32 distinct cellular subpopulations derived from the maternal-fetal interface.

Publication-ready Visualizations

Generate high-quality plots including DimPlot, AlluvialPlot, GroupPreferencePlot, VlnPlot, and FeaturePlot.

Flexible Gene Set Scoring

Run gene set scoring on 5,809 curated datasets using single or paired gene sets.

Reproducible Results

One-click access to datasets, consolidated downloads, and step-by-step case studies.

Comprehensive Data

73,025 cells and 25,858 genes spanning 28 distinct cell types from normal pregnancy and RSA samples.

User-friendly Interface

Intuitive web interface built with Shiny for easy navigation and analysis.

Usage

Prerequisites

  • R 4.5.0 or higher
  • Required R packages: shiny, htmltools, htmlwidgets, base64enc, Seurat, tidyverse, qs, gridExtra, ggpubr, ggplot2, reshape2, Nebulosa, DT

Installation

install.packages(c("shiny", "htmltools", "htmlwidgets", "base64enc", "Seurat", "tidyverse", "qs", "gridExtra", "ggpubr", "ggplot2", "reshape2", "Nebulosa", "DT"))

Running the Application

library(shiny)
runApp("app.R", host="0.0.0.0", port=3838)

The application will be available at http://localhost:3838

Deployment

Option 1: Shinyapps.io (Recommended)

Deploy your Shiny app to Shinyapps.io for easy sharing:

# Install rsconnect package
install.packages("rsconnect")

# Connect to your Shinyapps.io account
rsconnect::setAccountInfo(name='your-account', token='your-token', secret='your-secret')

# Deploy the app
rsconnect::deployApp("app.R")

Option 2: Docker

Use Docker to containerize and deploy your app:

# Build the Docker image
docker build -t rsadb .

# Run the Docker container
docker run -p 3838:3838 rsadb

Option 3: Local Server

Run the app on your own server using R:

Rscript -e "library(shiny); runApp('app.R', host='0.0.0.0', port=3838)"

About

Data Source

The single-cell RNA-seq data was collected from the maternal-fetal interface of both normal pregnancy (NP) and recurrent spontaneous abortion (RSA) samples. The data was processed using Seurat and normalized with standardized workflows.
